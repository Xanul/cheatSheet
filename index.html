Important HTML Elements

Document metadata

Tag Name Description
<head> Head Contains machine-readable information (metadata) about the document
<link> Link Specifies the relationships betweeen the current document and an external resource
<title> Title Defines the document title that is shown in the browsers title bar
<style> Style Contains style information for a document
<meta> Meta Represents metadata that cannot be represented by other HTML meta-related elemets

Content sectioning

Tag Name Description
<body> Body Represents the content of and HTML document, there can only be one
<header> Header Represents introductory content, typically a group of introductory or navigational elements
<footer> Footer Represents a content for its nearest sectioning content, typically contains information about the author or copyright
<aside> Aside Represent a portion of a document whose content is only indirectly related to the documents main content
<nav> Navigation Represents a section of a page whose purpose is to provide navigation links
